Backup Automático

O sistema realiza cópias de segurança automáticas todos os dias às 22:00. Esses arquivos são armazenados localmente e na nuvem para garantir a integridade dos seus dados.

Backup Manual

Restauração de Dados

Selecione um arquivo de backup anterior para restaurar o banco de dados do sistema.

Atenção: Ação Irreversível

Todos os dados atuais serão apagados e substituídos pelo conteúdo do backup selecionado. Certifique-se de que escolheu o arquivo correto.

* O sistema solicitará a senha de administrador para validar qualquer restauração.
